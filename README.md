# API
The Needy API

## Setup
```sh
python3 -m venv venv

# On windows
./venv/Scripts/activate

# On Linux
source venv/bin/activate

pip install -r requirements.txt

cd needy
python manage.py runserver
```

## Update

```
# On windows
./venv/Scripts/activate

# On Linux
source venv/bin/activate

pip install --upgrade -r requirements.txt # may fail once
```
